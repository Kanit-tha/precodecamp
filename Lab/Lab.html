style 		กำหนดตำแหน่งที่อยู่ของปลายทางที่จะนำไป หรือเปิดขึ้น
ลักษณะการเปิดหรือการแสดงหน้าต่าง

form action ตำแหน่งที่จะส่งข้อมูลไป
button style รายละเอียดของปุ่ม ประเภท
Differentiation
Tag ความเหมือน ความแตกต่าง
hr & br เป็น tag แบบ empty element hr เป็นคำสั่งสร้างเส้นแบ่ง
br เป็นคำสั่งขึ้นบรรทัดใหม่
ul & ol เป็นคำสั่งเกี่ยวกับ list หัวข้อของ list ul ลักษณะหัวจะเป็นสัญลักษณ์
ol ลักษณะหัวจะเป็นตัวเลข
select & checkbox & radio ทั้งสามเป็นคำสั่งเกี่ยวกับการเลือก ตัวเลือก select เป็น Tag คำลั่ง แสดงการเลือกตัวเลือกแบบ drop down เลือกได้เพียงคำตอบเดียว
radio เป็น Type ของ Tag input แสดงการเลือกแบบแสดงสัญลักษณ์ วงกลม เลือกได้เพียงคำตอบเดียว
checkbox เป็น Type ของ Tag input แสดงการเลือกแบบแสดงสัญลักษณ์ สี่เหลี่ยม เลือกได้หลายคำตอบ
div & span เป็นคำสั่งสร้างกล่องเหมือนกัน div สร้างออกมาเป็น Element แบบ block level
span สร้างออกมาเป็น Element แบบ inline level
script & style เป็นคำสั่งสำหรับภาษาอื่นในไฟล์ html script เป็นการเขียนคำสั่งที่เป็น javascript
style เป็นการเขียนคำสั่งที่เป็น css
Input Type
input type คำอธิบาย
button ลักษณะเป็นปุ่มกด
checkbox ลักษณะเป็นช่องสี่เหลี่ยม เมื่อคลิกจะเกิดเครื่องหมายขึ้น
date เป็นช่องหน้าต่างขึ้นเมื่อคลิก ให้เลือก เดือน วัน และปี
datetime เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
datetime-local เป็นช่องหน้าต่างขึ้นเมื่อคลิก 2 ส่วน ใหญ่ เดือนวันปี และเวลา
month เป็นช่องหน้าต่างขึ้นเมื่อคลิก ให้เลือก เดือนและปี
week เป็นช่องหน้าต่างขึ้นเมื่อคลิก ให้เลือก สัปดาห์ที่เท่าไหร่และปี
time เป็นช่องหน้าต่างขึ้นเมื่อคลิก ให้เลือก เวลา
color เป็นช่องสีเหลี่ยมแสดงสีดำเป็นค่าเริ่มต้น เมื่อคลิกปรากฏหน้าต่างให้เลือกสี
email เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
password เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้ เมื่อกรอกลงไปจะขึ้น Dot ไม่ให้เห็น
tel เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
text เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
number เป็นช่องสีเหลี่ยมว่างๆ ด้านขวาสามารถคลิกเพื่อเปลี่ยนตัวเลขได้
radio ลักษณะเป็นช่องวงกลม เมื่อคลิกจะเกิดเครื่องหมายขึ้น
range เป็นแภบเลื่อน สามารถเลื่อนไปซ้ายขวา เพื่อเลือกค่า
file เป็นปุ่ม คลิกเพื่อเลือกไฟล์ เมื่อคลิกจะเป็นหน้าต่างแฟ้มให้เลือกเอกสาร
image เป็นปุ่ม
url เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
search เป็นช่องสีเหลี่ยมว่างๆ สามารถกรอกของความลงไปได้
submit เป็นปุ่ม
reset เป็นปุ่ม
hidden ไม่ปรากฏ
